# Gemma 4 Heretic Agent

A full-featured local LLM agent with computer use, vision, and system execution capabilities.

## Features
- **Native Computer Control:** Click, type, and move the mouse.
- **Computer Vision:** Real-time screenshots processed by the model.
- **System Execution:** Direct Powershell command execution.
- **File Management:** Read, write, and list local files.
- **Modern Web UI:** Markdown rendering, code highlighting, and expandable reasoning boxes.
- **100% Local:** Powered by Ollama and Gemma 4 Heretic.
